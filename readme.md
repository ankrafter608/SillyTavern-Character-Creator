# SillyTavern Character Creator (Enhanced Fork)

This is an enhanced fork of the original [SillyTavern-Character-Creator](https://github.com/bmen25124/SillyTavern-Character-Creator).

## Основные возможности

Этот инструмент помогает создавать и редактировать карточки персонажей для SillyTavern с помощью ИИ. Основные отличия этого форка:

- **Интерактивный чат-помощник**: редактируйте персонажа в режиме реального времени, общаясь с ИИ.
- **Поддержка Lorebook**: создание и редактирование записей лорбука с встроенным счетчиком токенов.
- **Двухпанельный интерфейс**: поля персонажа слева, чат с ассистентом справа.
- **Автономный режим**: создание полного персонажа по одному краткому описанию.
- **Улучшенный экспорт**: поддержка JSON V3 и сохранение данных напрямую в SillyTavern.
- **База знаний**: возможность загружать файлы для анализа ИИ при создании персонажа.

## Установка

Установите через менеджер расширений SillyTavern, используя ссылку на этот репозиторий.
